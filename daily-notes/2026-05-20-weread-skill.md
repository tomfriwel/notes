# 微信读书skill

需要准备的：
1. 微信读书API Key
2. WorkBuddy

## 1. 微信读书API Key

https://weread.qq.com/r/weread-skills
底部微信扫码登录后，获取API Key，复制。

## 2. 安装WorkBuddy

https://www.codebuddy.cn/work/
下载安装

打开软件，左侧找到“技能”，搜“微信读书助手”，并安装。

也可以直接对话框里说：帮我安装微信读书skill。

我这里已经安装好了。

配置API Key，直接对话框问如何配置，跟着步骤走即可。

## 3. 使用

第一次对话会要求配置WEREAD_API_KEY，即第1步获取的API Key，根据提示配置好即可。

然后，就可以开始使用了。
