
`$ npm install hexo -g`

> install hexo globally

In project's dir:
`$ hexo init`

Or not have a dir:

`$ hexo init demo`

`$ npm install`

> install required modules

`$ hexo g`

`$ hexo s`

`$ hexo new post 'blog name'`

> create a new post

`$ npm install hexo-deployer-git --save`

> for git

`$ hexo d -g`

> deploy
